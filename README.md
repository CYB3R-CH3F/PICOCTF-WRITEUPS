## 📖 Write-Up Format

Every write-up follows this structure so it's easy to follow:

1. **What's the challenge asking?** — plain English, no jargon
2. **Concepts you need to know first** — explained from scratch
3. **Connection / file details** — exactly what was given
4. **Commands used** — with every flag and argument explained
5. **Mistakes I made** — so you don't make the same ones
6. **Flag captured** — the actual flag
7. **Key takeaways** — one lesson per point
