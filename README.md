Who is this for?
If you've never touched a terminal, never heard of SSH, or have no idea what a flag is in a CTF — this is for you.

Every write-up here breaks down exactly what the challenge is asking, what tools and concepts you need to know first, the exact commands used, mistakes made along the way, and what the flag looks like. No steps skipped. No assumed knowledge.

📖 Write-Up Format
Every write-up follows this structure so it's easy to follow:

What's the challenge asking? — plain English, no jargon
Concepts you need to know first — explained from scratch
Connection / file details — exactly what was given
Commands used — with every flag and argument explained
Mistakes I made — so you don't make the same ones
Flag captured — the actual flag
Key takeaways — one lesson per point
